Documentation API REST LeBonSandwich

  • /api/authentification Methode : POST

    Description : Permet de récupérer un token de sécurité
    Params : {username:root, password:root}
  • /api/categorieIngredient Methode : GET

    Description : Permet de récupérer tout les catégories des ingrédients
    Params : Pas de paramètres
  • /api/categorieIngredient/{id} Methode : GET

    Description : Permet de récupérer une catégorie d'ingrédients
    Params : Pas de paramètres
  • /api/categorieIngredient Methode : POST

    Description : Permet de créer une catégorie d'ingrédients
    Params : {nom:String, description:String}
  • /api/categorieIngredient/{categorieId}/ingredient Methode : GET

    Description : Récupére tous les liens d'ingrédient d'une catégorie
    Params : Aucun paramètre nécessaire
  • /api/categorieIngredient/{categorieId}/ingredients/{ingredientId} Methode : GET

    Description : Récupére un ingrédients d'une catégorie
    Params : Aucun paramètre nécessaire
  • /api/categorieIngredient/{categorieId}/ingredients Methode : POST

    Description : Ajoute un ingredient à une catégorie
    Params : {nom : String}
  • /api/categorieIngredient/{categorieId} Methode : DELETE

    Description : Supprime une catégorie
    Params : Aucun paramètre nécessaire
  • /api/commandes Methode : GET

    Description : Permet de récupérer toutes les commandes
    Params : Aucun paramètre nécessaire

    Description : Permet de valider une commande
    Params : Aucun paramètre nécessaire
  • /api/commandes/{commandeToken}/{commandeId}/update/{sandwichId} Methode : PUT

    Description : Permet de mettre à jour un sandwich d'une commande
    Params : Aucun paramètre nécessaire
  • /api/commandes Methode : POST

    Description : Ajoute une commande
    Params : {nom : String}
  • /api/commandes/{commandeId}/sandwich/{sandwichId} Methode : Delete

    Description : Supprime le sandwich d'une commande
